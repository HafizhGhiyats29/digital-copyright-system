# digital-copyright-system
tutorial clone sampe push
1. git clone https://github.com/HafizhGhiyats29/digital-copyright-system.git
2. cd digital-copyright-system
3. git checkout -b main
4. sebelum git add . , coba tambahin kode di format file .py mana saja contoh di file app.py tambahin code print("hello world")
5. git add .
6. git commit -m "tulis pesan perubahan"
7. git remote -v (buat pastiin dah terhubung sama link remotenya)
8. git push -u origin main
